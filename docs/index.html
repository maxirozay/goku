Getting started

To use Goku CSS you only need to install and import it.

Install with: yarn add -D goku-css or npm install --save-dev goku-css
Then import 'goku-css' or 'goku-css/dist/goku-legacy.css' for Internet Explorer

Customization

--primary-color: #444;

--primary-color-active: #888;
--secondary-color: #eee;
--secondary-color-text: #444;

--error-color: red;
--border-radius: 4px;
--border-width: 1px;

Spacing

You can try to add x, y, t, r, b or l before the size of the margin or padding to affect only some sides (e.g. my1, px1). For the margin, if you don't specify a size, like m or mx, the margin will be set to auto.
