Customization

You can modifiy any following CSS variable. You can do it with Javascript with something like document.documentElement.style.setProperty('--primary-color', '#FFFFFF').

--secondary-color: #444;
--success-color: #55ff44;
--error-color: red;
--border-radius: 4px;
--border-width: 1px;
--padding: 0.2rem 0.4rem 0.2rem 0.4rem;
--margin: 0.2rem 0.8rem 0.2rem 0;
--shadow: 0 0.1rem 0.1rem #bbb;
